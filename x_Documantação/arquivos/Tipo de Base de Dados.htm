Para armazenamento dos dados a serem exibidos no relatório, estão disponíveis dois tipos de bases de dados: ACCESS e MySQL.

     Por padrão, o ACCESS vem selecionado, todavia é possível alternar entre eles através do menu: Configurar > Avançado > Tipo de Base de Dados.

     Selecionando-se ACCESS, as configurações são mais simples e definidas automaticamente. Todavia é possível altera-las acessando Configurar Access.

     Selecionando-se MySQL, faz-se necessário a existência de um servidor de MySQL. As configurações do MySQL podem ser alteradas acessando Configurar MySQL.
